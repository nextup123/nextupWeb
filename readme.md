# 🚀 NextUp Web

A modern web application built with a focus on performance, scalability, and clean user experience.

---

## 🛠️ Getting Started

Follow these steps to run the project locally:

```bash
cd nextupWeb/
npm install
npm run dev
```

Once started, open your browser and navigate to:

```
http://localhost:3000
```

---

## 📂 Project Structure

```
nextupWeb/
├── config/              # Projejct related config files like path.js
├── controller/          # Backend - Controller layer
├── public/              # Frontend - All Pages
├── ros/                 # Ros Integration - using rclnodejs
└── routes/              # Backend - API Endpoints Layer
└── service/             # Backend - Service Layer
└── user_config/         # Helper functions
└── utils/               # Backend - Utility/Helper Functions
└── ws/                  # WebSocket Logic
└── package.json         # Helper functions
```
